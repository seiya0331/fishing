# Fishing

## サイト概要

### サイトテーマ

自分が釣った魚を投稿したり、他の人の投稿を閲覧したり、コメント欄で語りあったり出来るサイトです。

### テーマを選んだ理由

趣味の釣りで自分が釣った魚を共有したいと思い釣り専用のSNSがあればいいと考えました。
自分が使っている道具の紹介や釣りをしている場所の共有が出来たらもっと楽しく釣りができると思い制作しました。


### ターゲットユーザ
- 釣りが好きな人
- 最近釣りを始めた人
- 釣り仲間を探している人

### 主な利用シーン
- 魚を釣り上げて写真を投稿したい人
- 便利な釣り道具を紹介、探している人
